class Logger (name, datetime=None, use_csv=True, use_tensorboard=False)

Logger logs metrics to CSV files / tensorboard :name: logging name (e.g. model name / dataset name / …) :datetime: date and time of logging start (useful in case of multiple runs). Default: current date and time is picked :use_csv: log output to csv files (needed for plotting) :use_tensorboard: log output to tensorboard

Default: False :return: datetime, index (helpful, if datetime / index wasn't given)

Default: False :return: datetime, index (helpful, if datetime / index wasn't given)
